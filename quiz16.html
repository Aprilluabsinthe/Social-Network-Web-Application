Lecture #16 Quiz

  1. Save the HTML source from this Quiz #16 HTML page into a file called quiz16.html into the top-level directory of your class repo.
  2. Add and commit your quiz16.html file:
    • git add quiz16.html
    • git commit -m "Download of quiz16.html, before modification."
  3. Modify the <style> element in the <head> section of your quiz16.html so that the <div>, below, is responsive:
    • The <div> width must scale (grow/shrink) when the viewport width changes
    • The font-size of the letters in the <div> must scale when the viewport width changes
  4. Optional: Make everything else on this page responsively scale, as well.
  5. After making the required changes to your quiz16.html file, add, commit, and push your changes to turn in this quiz:
    • git add quiz16.html
    • git commit -m "Working quiz16 submitted."
    • git push
